# User Details React App

This React.js application fetches user details from the [Random User Generator API]([https://randomuser.me/](https://randomuser.me/api/?page=1&results=1&seed=abc)) and displays them in a responsive and visually appealing manner.

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) installed on your machine

### Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/user-details-react-app.git
cd user-details-react-app
```

2. Install dependencies:

```bash
npm install
```

## Usage

1. Run the development server:

```bash
npm start
```

2. Open your browser and navigate to [http://localhost:3000](http://localhost:3000) to view the app.
